================================================================================
■ KeyReader ライブラリ関数一覧（リファレンス）
================================================================================

【1. 設定用関数（setup関数内で begin() の前に呼び出す）】

● reader.row(num);
  - 引数：int (行数)
  - 説明：キーパッドの行数を設定します（標準は4）。

● reader.col(num);
  - 引数：int (列数)
  - 説明：キーパッドの列数を設定します（標準は4）。

● reader.passLong(len);
  - 引数：int (最大桁数)
  - 説明：入力できる最大文字数を指定します（これを超えると入力が無視されます）。

● reader.buzzSound(pin, freq);
  - 引数：int (ピン番号), unsigned int (周波数[Hz])
  - 説明：ブザーの接続ピンと、通常入力時の音の高さを設定します。
         ※クリア時は自動的にこの半分の周波数（低い音）になります。

● reader.setRowPins(pins);
  - 引数：int配列[]
  - 説明：行に接続するArduinoのピン番号を配列でまとめて渡します。

● reader.setColPins(pins);
  - 引数：int配列[]
  - 説明：列に接続するArduinoのピン番号を配列でまとめて渡します。

● reader.enterButton(pin);
  - 引数：int (ピン番号)
  - 説明：決定（エンター）ボタンのピン番号を設定します。

● reader.deleteButton(pin);
  - 引数：int (ピン番号)
  - 説明：全消し（クリア）ボタンのピン番号を設定します。

● reader.arrange(map);
  - 引数：char型の2次元配列 [4][4]
  - 説明：物理的なキー配置の文字マップをライブラリに登録します。


【2. 初期化関数（setup関数内の最後に必ず1回だけ呼び出す）】

● reader.begin();
  - 引数：なし
  - 説明：設定された情報をもとに、内部で pinMode や初期HIGH出力を自動で一括実行します。


【3. 実行関数（loop関数内で常に呼び出す）】

● reader.read(targetVariable);
  - 引数：String型変数（ユーザーがあらかじめ用意した変数）
  - 説明：バックグラウンドで常にキーパッドを高速スキャンします。
         決定ボタンが押されて入力が「確定」した瞬間だけ true を返し、
         引数に渡したString型変数の中身を入力された文字列に書き換えます。

================================================================================
