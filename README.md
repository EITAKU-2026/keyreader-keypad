# 特徴
1. オールインワンのキーパッド入力の管理
   キーのスキャンから指定桁数までの自動蓄積、配列のオーバーフロー防止までをライブラリ側が全自動で行います。
2. 独立した物理ボタンのサポート
   キーパッド内のキーだけでなく、外部に接続した独立のボタン（決定・クリア）をサポートしています。ノンブロッキングのチャタリング防止（デバウンス）付きで誤作動を起こしません。
3. 3つの音の自動鳴り分け
   ユーザーが基本の周波数を1行指定するだけで、通常入力（ユーザー指定の音）、クリア（指定の半分の低音）、決定（指定の2倍の高音）をライブラリが自動計算して気持ちよく鳴り分けます。
4. 完全ノンブロッキング設計
   delay() を一切使用せず、すべて millis() で時間管理をしているため、1秒間に何千回もの超高速スキャンを実現しています。他の処理（モーターや液晶制御）を邪魔せず、キーの取りこぼしもありません。



# 接続参考例
* RowPins (行) : Arduino D9, D8, D7, D6  -> キーパッドの行へ
* ColPins (列) : Arduino D5, D4, D3, D2  -> キーパッドの列へ
* EnterButton  : Arduino D11 ------------> 決定ボタン (タクトスイッチ) -> GND
* ClearButton  : Arduino D12 ------------> クリアボタン (タクトスイッチ) -> GND
* Buzzer       : Arduino D10 ------------> 圧電ブザー -> GND
