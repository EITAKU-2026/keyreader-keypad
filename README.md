# 主な強み
1. オールインワンのキーパッド入力の管理
   キーのスキャンから指定桁数までの自動蓄積、配列のオーバーフロー防止までをライブラリ側が全自動で行います。
2. 独立した物理ボタンのサポート
   キーパッド内のキーだけでなく、外部に接続した独立のボタン（決定・クリア）をサポートしています。ノンブロッキングのチャタリング防止（デバウンス）付きで誤作動を起こしません。
3. 3つの音の自動鳴り分け
   ユーザーが基本の周波数を1行指定するだけで、通常入力（ユーザー指定の音）、クリア（指定の半分の低音）、決定（指定の2倍の高音）をライブラリが自動計算して気持ちよく鳴り分けます。
4. 完全ノンブロッキング設計
   delay() を一切使用せず、すべて millis() で時間管理をしているため、1秒間に何千回もの超高速スキャンを実現しています。他の処理（モーターや液晶制御）を邪魔せず、キーの取りこぼしもありません。



# 接続参考例
* RowPins (行) : Arduino D9, D8, D7, D6  -> キーパッドの行へ
* ColPins (列) : Arduino D5, D4, D3, D2  -> キーパッドの列へ
* EnterButton  : Arduino D11 ------------> 決定ボタン (タクトスイッチ) -> GND
* ClearButton  : Arduino D12 ------------> クリアボタン (タクトスイッチ) -> GND
* Buzzer       : Arduino D10 ------------> 圧電ブザー -> GND

# 使用例


```cpp
#include <KeyReader.h> // <> で囲んで呼び出してください。

KeyReader reader;    // ライブラリの実体（オブジェクト）を作成します。
String keyPassword;  // 入力された文字列を受け取るためのユーザー変数です。名前は任意に決められます。

// キー配置の定義をしてください。
char myMap[4][4] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'F','0','E','D'}
};

void setup() {
  Serial.begin(9600);

  // 1. 各種カスタマイズ設定
  reader.row(4);
  reader.col(4);
  reader.passLong(20);
  reader.buzzSound(10, 1000); // 10番ピン、1000Hz
  
  int rPins[] = {9, 8, 7, 6};
  reader.setRowPins(rPins);
  
  int cPins[] = {5, 4, 3, 2};
  reader.setColPins(cPins);
  
  reader.enterButton(11);
  reader.deleteButton(12);
  reader.arrange(myMap);

  // 2. 最後に必ず初期化を呼んでください。pinMode設定などが自動実行されます。
  reader.begin();
  
  Serial.println(F("KeyReader Ready."));
}

void loop() {
  // 3. ループ内で常に呼び出す
  // 決定ボタンが押されて「確定」した瞬間だけ、if文の中身が実行されます。
  if (reader.read(keyPassword)) {
    Serial.println();
    Serial.print(F("確定パスワード: "));
    Serial.println(keyPassword); // ユーザーが定義した変数に入力した文字列を代入して表示します。
  }
}

```
